    Im using Jellyfin#

    So im using Jellyfin now for a long Time with Jellyseerr, but i now wanted to Tell my Experience with Jellyfin and the Other Services with Jellyfin.

    How to install#

    First of all, how to install it because thats the Point of my Blog, to tell how i do it.

    Backend#

    Before you can do the Frontend you need the Backend (You see later why) So first we install some Tools, the Tools that we need are: Jackett, Radarr, Sonarr and Transmission. Before we install it i want to give a Disclaimer becuase Yes it is Piracy but in my Country its Legal so please first look if its Legal in your Country.

    CasaOS#

    So now lets install it, i use CasaOS for the Docker UI so its easier, when you install CasaOS on a Proxmox Server like me then i can recommend this website to you: helper-scripts.com There are some good scripts, but if you gonna install it Bare Metal then you need to Install Debian (12 i recommend) and then use the Script “curl -fsSL https://get.casaos.io | sudo bash” but i dont know the Procedure for Bare Metal.

    After you installed CasaOS you need to first configure. Just Type the Username, Password and the Password again and then you Good to go. Then you need to first Update the CasaOS, for this you need to go on the Second Icon left in the Navbar and then Update. In The App Store we need to add a Repo for the Jellyfin App so we go to the App Store -> More Apps -> Paste “https://github.com/bigbeartechworld/big-bear-casaos/archive/refs/heads/master.zip" -> Add -> You now Ready to Start

    Jackett and Transmission#

    First we install Transmission and Jackett, thats so easy, i think you should do it by yourself. After you finish installing it, you go in CasaOS to Transmission and on the Three Dots and then Settings. Unter Volumes you do it like me: ![[Pasted image 20241224131200.png]] Every Folder is already there so you dont need to manually add them, save it and then we go in to Jackett

    Configure Jackett#

    Just Ignore the Warning with the Old version, as long you dont Publish it, its safe enough. We need a Indexer so we go to Add Indexer and Search for a Good Indexer, i recommend Pirate Bay but its up to you what you use. After you selected it you can use add Selected and we are done with Jackett and Transmission

    Configure Transmission (Optional)#

    If you want to really sure that you dont do anything Illegal with republish things you go in your Transmission (Password and User are casaos) and then on the Burger in the upper right, there you go in Speed and there you set your Upload to Zero.

    Sonarr and Radarr#

    to Install these in your CasaOS you just do the Same thing like with Jackett and Transmission. Then we go again in the Settings from Sonarr and Radarr to set the Tag to latest and Save.

    Configure Sonarr#

    We gonna Start with Sonarr, just open it and then you can do the Authentication like you want. After that go to the Settings -> Profiles and Delete every Profile you dont need, i recomment HD-720p or HD-1080p (but if you want you can delete all of them and add a new one)

    Next we gonna go to Indexers and add a new one -> Torznab and then do it like me: ![[Pasted image 20241224132721.png]] Save it and go to Download Clients -> Add a new one with Transmission and then again like me: ![[Pasted image 20241224132925.png]] Then we are gonna go to Media Management and add a Root Folder with the path “tv” Save and we are Finished with Sonarr FOR NOW

    Configure Radarr#

    Same Thing like Sonarr Settings -> Profiles Settings -> Indexers Settings -> Download Clients Settings -> Media Management (But with Movies)

    Frontend#

    Jellyfin#

    WOHOO we are going to the Frontend now. But first we need to Install Jellyfin, the actual Thing we are gonna Watch on it. Installation is like everything else. Again in the Settings we need to change the Tag to latest and Save. Go through the wizard like you want until the Media Library, add Movie with the Path /Media/Movies and Serie with the Path /Media/TV Shows then just go through. When you Finished then we are Finished with Jellyfin.

    Jellyseerr#

    And Last but not Least we are gonna put everything together with Jellyseerr. Again Install it and then set the Tag to latest. Then Open it and do it like i did.

    Click “Configure Jellyfin” -> Type in " Jellyfin URL (Internal IP with Port (mostly 8097)) E-Mailadress (it must not Assosciated with Jellyfin) Username (from the Admin Jellyfin) Password " -> Sign In -> Click “Sync Libraries” and add every Library -> Continue -> Add a Radarr and Sonarr Server and set them to Default Server (You find the API Key under Settings -> General -> API Key) -> Finish the Setup and you can now Request and automatically Download them

    My Experience#

    I Think its a really good Platform for everything that you need when you just want to Watch Movies and Series and with Jellyseer you can automatically Download Movies and Series and so its just Perfect for what i need it and i can give Friend a Account and they can just watch also my Movies and if you do it Right you can add a OAuth.
